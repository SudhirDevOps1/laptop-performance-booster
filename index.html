1. %temp% Folder Clean Karna

Temporary Files Delete Karo

ü§î Ye Kya Hota Hai?

%temp% ek environment variable hai jo temporary files folder ko point karta hai. Jab bhi aap koi software install karte ho, ya koi program run karta hai, wo apni temporary files yahan store karta hai.

Location: C:\Users\YourUsername\AppData\Local\Temp

üìÅ Isme Kya Files Hoti Hain?

  • ‚Ä¢ Software installation temp files
  • ‚Ä¢ Browser downloads ke temp files
  • ‚Ä¢ Application cache files
  • ‚Ä¢ Log files

‚úÖ Method 1: Run Dialog

Step 1 Windows + R press karo
Step 2 Type karo: %temp%
Step 3 Enter press karo
Step 4 Ctrl + A (Select All)
Step 5 Shift + Delete (Permanent Delete)

‚úÖ Method 2: Command Prompt (CMD)

del /q/f/s %TEMP%\*

üîç Command Explanation:

/q Quiet mode - confirmation nahi mangega
/f Force delete - read-only files bhi delete
/s Subfolders mein bhi delete kare
‚ö°

Expected Result:

500MB se 5GB tak space free ho sakti hai!

ü§î Ye Kya Hota Hai?

Location: C:\Windows\Prefetch

‚úÖ Method 1: Run Dialog

1. Win + R press karo

2. Type: prefetch

3. Admin permission do (Yes click karo)

4. Ctrl + A ‚Üí Shift + Del

‚úÖ Method 2: CMD (Admin)
